# 一个简单的外卖系统

## 仓库简介

项目源码来自于黑马在哔站拥有著名的拥有百万播放的苍穹外卖系统
本仓库为作者在跟随视频学习期间自主理解并适当更改的代码，可以正常运行

由于前端是展示型页面，主要用于测试接口以及服务的可行性，
而且利用Postman等工具时需要构建数据体，所以在此省略
项目主要是后端代码，用的是SpringBoot语言
是一个非常好的练手的项目

## 项目介绍

黑马程序员Java项目实战《苍穹外卖》，最适合新手的SpringBoot+SSM的企业级Java项目实战
